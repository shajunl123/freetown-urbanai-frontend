> **编制说明**：本方案基于典型别墅施工场景编制，所有技术参数均基于合理假设（详见各章节"假设条件"）。实际施工前须根据岩土工程勘察报告、施工图设计文件及现场实际条件进行复核调整。标注 **【待定项】** 的内容须由项目技术负责人根据实际情况填写确认。

#### 1.2 假设条件

> ⚠️ **以下为典型场景假设，实际施工前必须核对：**
> - 土层分布：0~0.5m杂填土 → 0.5~2.0m粉质粘土 → 2.0~5.0m粘土 → 5.0m以下强风化基岩
> - 地下水位：枯水期-4.5m，丰水期-3.8m（基坑底板在枯水期水位以上）
> - 场地周边：北侧距相邻别墅约6m，东侧距市政道路约8m，南侧、西侧空旷
> - 施工季节：非雨季

### 2. 开挖原则

1. **分层、分段、对称、限时**——控制基坑变形，保护周边环境
2. **先护后挖**——围护结构施工完成并达到设计强度后方可开挖
3. **严禁超挖**——每层开挖深度不超过围护设计允许值
4. **随挖随撑**——开挖到支撑标高后及时施工支撑或垫层
5. **信息化施工**——根据监测数据动态调整开挖节奏

- [ ] 复核岩土工程勘察报告，确认土层分布及地下水位

- [ ] 完成施工技术交底

#### 3.2 现场准备

- [ ] 场地平整，清除地表障碍物
- [ ] 测量放线，标定基坑边线、开挖边线、坡顶线
- [ ] 设置基坑变形监测点（坡顶水平位移、沉降、周边建筑沉降）
- [ ] 施工临时道路硬化，确保重载车辆通行
- [ ] 土方运输车辆及路线确认，办理渣土运输许可
- [ ] 基坑周边截水沟/挡水墙施工

#### 3.3 降水准备（如需要）

> **【待定项：根据地下水位和开挖深度关系决定是否降水】**

- 假设条件：基坑底标高-3.5m，丰水期水位-3.8m，旱季施工，水位在底板以下
- **结论：本方案假设条件下不需要降水，如丰水期施工或水位高于底板，须增加降水方案**

如需降水，常见方案：
| 方案 | 适用条件 | 优缺点 |
|------|----------|--------|
| 坑内明排 | 水量小、渗透性低 | 简单便宜，但排水不彻底 |
| 轻型井点 | 水位高、土层渗透性中等 | 降水效果好，需要设备投入 |
| 管井降水 | 水量大、降水深度大 | 效果好，成本较高 |

### 4. 开挖方案

#### 4.1 开挖分区

基坑开挖分为两个阶段：

**第一阶段：表层清表（0 ~ -0.5m）**
- 机械开挖，清除杂填土、建筑垃圾、树根等
- 清表土方外运，不得用于回填

**第二阶段：主体开挖（-0.5m ~ -3.5m）**
- 分层开挖，每层深度不超过1.0m
- 共分3层：-0.5m → -1.5m → -2.5m → -3.5m（基底）

#### 4.2 开挖顺序

```
        ┌──────────────────────────────┐
        │                              │
        │    ② 第二层（-1.5m~-2.5m）    │
        │                              │
        │    ③ 第三层（-2.5m~-3.5m）    │
        │         ↓ 基底               │
        └──────────────────────────────┘
        
        ① 先开挖四周，留核心土
        ② 再开挖中部
        ③ 最后修整基底
```

**具体步骤：**

1. **放坡开挖第一层（-0.5m ~ -1.5m）**
   - 四周边坡按1:0.5放坡（视土质调整）
   - 开挖宽度：基坑边线外扩工作面0.8m
   - 机械开挖至-1.5m，人工配合修坡

2. **放坡开挖第二层（-1.5m ~ -2.5m）**
   - 继续按1:0.5放坡
   - 每开挖一段（不超过20m），立即进行坡面防护
   - 坡面防护：挂Φ6@200钢筋网片 + 喷射C20混凝土，厚度80mm

3. **开挖第三层至基底（-2.5m ~ -3.5m）**
   - 最后一层开挖需人工配合，严禁机械扰动基底土
   - 基底预留200~300mm人工清底
   - 清底后立即施工垫层（C15素混凝土，厚度100mm）

4. **基底验收**
   - 检查基底标高、平整度
   - 检查有无软弱土层、异常情况
   - 验收合格后24小时内施工垫层

#### 4.3 土方运输

| 项目 | 要求 |
|------|------|
| 运输时间 | 晚22:00~早6:00禁止运输（视当地规定调整） |
| 车辆要求 | 密闭运输，车轮冲洗后出场 |
| 堆放要求 | 场内临时堆土高度不超过1.5m，距坑边不小于1.5倍开挖深度 |
| 弃土场地 | **【待定项：确认合法弃土场位置】** |

### 5. 基底处理与垫层

#### 5.1 基底暴露时间控制

- 基底清底完成后，**4小时内**必须施工垫层
- 如遇降雨，基底不得浸泡，立即覆盖彩条布+排水
- 基底被水浸泡后，必须清除浸泡土层，重新验收

#### 5.2 垫层施工

| 项目 | 参数 |
|------|------|
| 垫层混凝土 | C15素混凝土 |
| 垫层厚度 | 100mm |
| 垫层范围 | 基础轮廓外扩100mm |
| 施工要求 | 随浇随抹平，不留施工缝 |

### 6. 基坑监测

#### 6.1 监测项目

| 监测项目 | 监测频率 | 报警值 | 监测方法 |
|----------|----------|--------|----------|
| 坡顶水平位移 | 开挖期间1次/天 | 累计30mm或速率3mm/d | 全站仪 |
| 坡顶沉降 | 开挖期间1次/天 | 累计20mm | 水准仪 |
| 周边建筑沉降 | 开挖期间1次/天 | 累计15mm | 水准仪 |
| 地下水位 | 1次/天 | — | 水位管 |
| 裂缝监测 | 发现裂缝后2次/天 | 宽度>1.5mm | 裂缝观测仪 |

#### 6.2 监测点布置

- 坡顶：沿基坑周边每隔15~20m设一个监测点
- 周边建筑：在相邻建筑四角各设一个沉降监测点
- 地下水位：基坑内设1~2个水位观测管

#### 6.3 应急响应

| 预警等级 | 触发条件 | 响应措施 |
|----------|----------|----------|
| 黄色预警 | 位移达报警值70% | 加密监测，分析原因 |
| 橙色预警 | 位移达报警值90% | 暂停开挖，采取加固措施 |
| 红色预警 | 位移超过报警值 | 立即停工，组织专家论证，必要时回填反压 |

### 7. 安全文明施工

#### 7.1 安全措施

- 基坑周边设置1.2m高防护栏杆+踢脚板
- 基坑边缘1.5m范围内禁止堆载
- 重型车辆（混凝土罐车、泵车）行走路线距坑边不小于2倍开挖深度
- 基坑内作业人员必须佩戴安全帽，上下基坑走专用通道（禁止攀爬边坡）
- 雨后复工必须重新检查边坡稳定性

#### 7.2 扬尘控制

- 场地出入口设置自动冲洗设备
- 裸露土方及时覆盖（超过48小时未施工的区域）
- 基坑周边设置喷淋降尘系统（视当地要求）
- 渣土运输车辆必须密闭

## 第二篇：基坑围护方案

### 1. 围护方案选型

#### 1.1 方案比选

| 方案 | 适用条件 | 本项目适用性 | 造价参考 |
|------|----------|--------------|----------|
| **自然放坡+挂网喷浆** | 场地开阔、土质较好、开挖深度≤5m | ✅ 适用（本方案首选） | 80~150元/㎡ |
| 拉森钢板桩 | 场地受限、需止水、开挖深度≤6m | ⚠️ 可用但成本偏高 | 200~350元/㎡ |
| SMW工法桩 | 深基坑、需止水、周边环境敏感 | ❌ 本项目不需要 | 400~600元/㎡ |
| 灌注桩排桩+止水帷幕 | 深基坑、周边建筑距离近 | ❌ 本项目不需要 | 600~1000元/㎡ |

#### 1.2 选型结论

> **本项目基坑开挖深度3.5m，场地周边条件较好（最近建筑距离≥6m），土质为粘性土，具备放坡条件。**
>
> **推荐方案：自然放坡 + 挂网喷浆护坡**
>
> **【待定项：如实际地质条件为软弱土层或地下水位高于基底，须改用钢板桩或其他支护方案】**

---

### 2. 围护设计方案

#### 2.1 放坡参数

| 参数 | 取值 | 依据 |
|------|------|------|
| 综合放坡系数 | 1:0.5~1:0.75 | 粘性土，临时边坡 |
| 分级高度 | 每级≤1.5m | |
| 分级平台宽度 | 0.5m | |
| 坡面防护 | 挂网喷浆 | |

**放坡剖面示意：**

```
    地面 ±0.000
    ──────────────────────────────
     \  1:0.5
      \
       \───── 0.5m平台（-1.5m）
        \
         \  1:0.5
          \
           \───── 基底（-3.5m）
           
    ← 6m → 相邻建筑
```

#### 2.2 挂网喷浆设计

| 项目 | 参数 |
|------|------|
| 钢筋网 | Φ6@200×200，双向 |
| 加强筋 | Φ12@1500（水平向） |
| 锚钉 | Φ12钢筋，L=1.0m，间距1.5m×1.5m，梅花形布置 |
| 喷射混凝土 | C20，厚度80~100mm |
| 喷射范围 | 坡面全高度+坡顶外扩0.5m |

#### 2.3 施工工艺流程

```
坡面修整 → 清理浮土 → 打设锚钉 → 挂钢筋网片 → 
焊连接（网片搭接≥200mm） → 喷射混凝土 → 养护
```

**喷射混凝土施工要求：**
- 分层喷射，每层厚度30~50mm
- 喷射方向与坡面垂直
- 喷射完成后2小时内开始养护
- 养护时间不少于7天

#### 2.4 坡顶截水沟

| 项目 | 参数 |
|------|------|
| 截水沟位置 | 坡顶线外扩0.5m |
| 截面尺寸 | 300mm×300mm（宽×深） |
| 材料 | C20混凝土现浇或砖砌抹灰 |
| 排水方向 | 引至场地低点，接入市政雨水管网 |

---

### 3. 特殊部位处理

#### 3.1 基坑阳角

- 阳角部位应力集中，放坡系数放大至1:1
- 喷射混凝土厚度增加至120mm
- 锚钉间距加密至1.0m×1.0m

#### 3.2 基坑阴角

- 阴角部位排水不畅，设置集水坑（500mm×500mm×500mm）
- 坡面喷射混凝土延伸至阴角内侧各0.5m

#### 3.3 基坑出土口

- 出土口设置专用坡道，坡度≤1:6
- 坡道两侧设置防护栏杆
- 坡道面层铺设200mm厚碎石+钢板，确保车辆通行

---

### 4. 围护结构拆除

> **【待定项：本项目基坑围护为临时结构，主体结构完成后需拆除】**

#### 4.1 拆除时机

- 地下室侧墙防水施工完成
- 地下室顶板结构完成并达到设计强度
- 回填土施工至围护结构顶部以下0.5m

#### 4.2 拆除顺序

```
拆除顺序：自上而下，分段拆除

1. 拆除坡面喷射混凝土（机械破碎+人工配合）
2. 拔除锚钉（如采用可回收锚钉）
3. 清除钢筋网片
4. 拆除分级平台
5. 分层回填并压实
```

#### 4.3 注意事项

- 拆除期间不得影响已完工的地下室结构
- 拆除的混凝土块及时清运，不得堆放在基坑边
- 如采用钢板桩支护，拔桩后立即回填桩孔，防止周边土体沉降

### 3. 拆模施工工艺

#### 3.1 拆模顺序

```
拆模总原则：后支先拆，先支后拆；先拆非承重部位，后拆承重部位

具体顺序：
1. 柱侧模 → 墙侧模 → 梁侧模（先拆侧模，后拆底模）
2. 梁底模 → 板底模（从跨中向两端对称拆除）
3. 支撑体系（自上而下逐层拆除）
```

#### 3.2 拆模操作要点

**墙体侧模拆除：**
1. 混凝土强度达到1.2MPa（手指按压无明显痕迹）
2. 先松动对拉螺栓和柱箍
3. 用撬棍轻轻松动模板，禁止硬撬
4. 拆除后立即清理模板表面，涂刷脱模剂备用

**顶板底模拆除：**
1. 确认同条件试块强度达到设计要求
2. 从跨中开始，向两端对称拆除
3. 先拆模板，再拆方木，最后拆钢管支撑
4. 拆除区域设置临时支撑（保留养护支撑），养护支撑间距不大于3m

**梁底模拆除：**
1. 跨度≤8m，强度达到75%方可拆除
2. 跨度>8m，强度达到100%方可拆除
3. 拆除时先松动扣件，用临时支撑托住梁底
4. 对称拆除，防止梁体偏载

#### 3.3 拆模安全要求

- 拆模区域下方设置警戒线，禁止人员通行
- 拆模人员佩戴安全帽、防砸鞋
- 模板传递禁止抛掷，使用绳索或传递工具
- 拆除的模板及时清理、归堆，不得堆放在楼板上集中荷载过大

### 4. 回填方案

#### 4.1 回填条件

| 条件 | 要求 |
|------|------|
| 回填时间 | 地下室结构验收合格后 |
| 防水要求 | 地下室侧墙防水层施工完成并验收 |
| 回填范围 | 地下室侧墙外侧至基坑边 |
| 回填深度 | 从基底至室外地坪（约3.5m） |

#### 4.2 回填材料

| 材料 | 要求 | 适用部位 |
|------|------|----------|
| 素土（粘土或粉质粘土） | 有机质含量≤5%，不含大块石、冻土、建筑垃圾 | 主要回填材料 |
| 灰土（3:7或2:8） | 石灰消解后过筛，与土拌合均匀 | 基底附近200mm范围 |
| 碎石土 | 最大粒径≤50mm | 排水层（如需要） |

**【待定项：如设计有特殊要求（如灰土回填、级配砂石回填），按设计执行】**

> ⚠️ **回填土不得使用：淤泥、淤泥质土、冻土、膨胀土、有机质土、建筑垃圾**

#### 4.3 回填施工工艺

```
基底清理 → 防水验收 → 分层摊铺 → 碾压/夯实 → 检测压实度 → 下一层
```

#### 4.4 分层参数

| 参数 | 取值 | 依据 |
|------|------|------|
| 每层虚铺厚度 | 250~300mm（机械碾压） / 200mm（人工夯实） | GB50202-2018 |
| 压实系数 | ≥0.94（一般要求） / ≥0.97（重要部位） | **【待定项：按设计要求】** |
| 压实方法 | 蛙式打夯机 / 振动压路机（场地允许时） | |

#### 4.5 回填施工要点

1. **对称回填**——地下室两侧同时回填，高差不超过500mm，防止侧墙受偏压
2. **分层压实**——每层摊铺后碾压或夯实，检测合格后方可摊铺下一层
3. **接茬处理**——每层回填接茬处做成阶梯形，阶梯宽度不小于高度的1.5倍
4. **管道保护**——回填至管道标高时，管道两侧对称人工夯实，防止管道位移
5. **回填至室外地坪**——顶板以上500mm范围内不得使用重型碾压设备

#### 4.6 压实度检测

| 检测项目 | 检测频率 | 检测方法 | 合格标准 |
|----------|----------|----------|----------|
| 压实系数 | 每层每50m²不少于1点 | 环刀法 / 灌砂法 | ≥设计要求值 |
| 含水率 | 与压实度同时检测 | 酒精燃烧法 | 最优含水率±2% |

> **每层回填压实度检测合格后方可进行下一层回填**

---

### 5. 回填施工顺序

#### 5.1 地下室侧墙外回填

```
第一步：地下室底板外侧（-3.5m ~ -3.0m）
  → 灰土回填200mm（基底附近）
  → 素土分层回填至-3.0m
  → 压实度检测

第二步：侧墙中部（-3.0m ~ -1.5m）
  → 素土分层回填
  → 每层300mm，碾压夯实
  → 压实度检测

第三步：侧墙顶部（-1.5m ~ ±0.000）
  → 素土分层回填
  → 室外地坪以下500mm范围采用人工夯实
  → 压实度检测
```

#### 5.2 室内地回填（如需要）

> **【待定项：地下室室内是否需要回填至设计标高，按设计要求】**

- 室内回填在顶板施工前完成（如采用架空板方案则不回填）
- 回填至设计标高后施工地面垫层

### 6. 回填质量控制

#### 6.1 常见质量问题及预防

| 质量问题 | 原因 | 预防措施 |
|----------|------|----------|
| 回填土下沉 | 虚铺过厚、压实不足、含水率不当 | 严格控制虚铺厚度，检测含水率 |
| 侧墙位移 | 单侧回填过高 | 两侧对称回填，高差≤500mm |
| 管道变形 | 管道两侧未对称夯实 | 管道两侧人工对称夯实 |
| 表面松散 | 含水率过低 | 洒水润湿后重新碾压 |
| 橡皮土 | 含水率过高 | 翻晒或换土处理 |

#### 6.2 验收标准

| 检查项目 | 允许偏差 | 检查方法 |
|----------|----------|----------|
| 回填标高 | ±50mm | 水准仪 |
| 表面平整度 | ±20mm | 2m靠尺 |
| 分层厚度 | ±50mm | 插钎法 |
| 压实系数 | ≥设计值 | 环刀法/灌砂法 |
